## Dataset

This project uses the **Network Intrusion Dataset** available on Kaggle.

**Dataset Link:**
https://www.kaggle.com/datasets/chethuhn/network-intrusion-dataset

The dataset contains network traffic records representing both benign and malicious activities. Multiple CSV files are combined into a unified dataset before preprocessing and model training.

> **Note:** The dataset is not included in this repository due to its size. Please download it from the Kaggle link above and place the CSV files in the `dataset/` directory before running the notebook.
